Telemetry
=========

The ``restgdf[telemetry]`` extra integrates with
`OpenTelemetry <https://opentelemetry.io/>`_ for distributed tracing and
log correlation. Install it alongside the base package:

.. code-block:: bash

   pip install "restgdf[telemetry]"

See :doc:`recipes/observability` for a practical integration guide.
